# Data Cleaning and Visualization Project

## Objective

The objective of this project is to clean a raw dataset and create visualizations to gain meaningful insights.

## Dataset

Titanic Dataset

## Tools Used

* Python
* Pandas
* Matplotlib

## Data Cleaning Steps

1. Loaded the dataset using Pandas.
2. Checked for missing values.
3. Removed duplicate records.
4. Filled missing values in age and embarked columns.

## Visualizations Created

1. Survival Count
2. Passenger Class Distribution
3. Gender Distribution

## Key Insights

* Most passengers did not survive.
* Third class had the highest number of passengers.
* Male passengers were more than female passengers.

## Conclusion

The dataset was successfully cleaned and analyzed. Visualizations helped understand passenger demographics and survival patterns.
